Overall Match displays an overall assessment which includes a weighted average of all of the displayed measures.
Importance: Olympic level archers want this within +/- 2.5%. Elite level archers want this within +/- 5%. Intermediate level archers want this within +/- 10%. Beginner level archers want this within +/- 15%.

Brace Height Variance displays the calculated optimal brace height in relation to the current brace height.
Importance: An optimal brace height reduces left/right grouping size significantly.
100% is Optimum. Above 100% suggests raising brace height. Below 100% suggests lowering brace height.
Nock Clearance displays the calculated amount of nock clearance as the nock is passing the cushion plunger or arrow rest.
Importance: Good clearance is necessary to optimize left/right arrow grouping.
